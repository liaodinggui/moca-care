## 四、交互说明

### 4.1 用户端交互

| 页面 | 交互动作 | 说明 |
|------|----------|------|
| 登录页 | 点击微信登录 | 调用微信授权，获取用户信息 |
| 登录页 | 选择角色 | 弹窗选择客户/商家角色 |
| 我的 Tab | 点击状态卡片 | 跳转对应状态的订单列表 |
| 订单详情 | 显示二维码 | 仅"使用中"状态订单显示 |
| 订单详情 | 点击支付 | 调起微信支付 |

### 4.2 商家端交互

| 页面 | 交互动作 | 说明 |
|------|----------|------|
| 商家首页 | 点击扫码核销 | 调起小程序扫一扫 |
| 扫码结果 | 选择核销数量 | 不能超过剩余数量 |
| 扫码结果 | 确认核销 | 提交核销，显示结果弹窗 |
| 订单管理 | 点击 Tab 筛选 | 切换订单状态筛选 |
| 订单管理 | 点击核销按钮 | 跳转手工核销页 |
